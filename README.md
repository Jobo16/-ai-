# 批量调用api工具

## 该工具通过SiliconCloud API将中文考试题目解析批量翻译成越南语，专为教育场景设计。

### 主要功能:
1. 集成SiliconCloud API进行翻译
2. 批量处理CSV文件中的中文内容
3. 生成专业的教育解析内容(正确答案解释+干扰选项分析)
4. 异步并发处理提高效率

### 使用方法:
1. 配置API密钥和输入文件路径
2. 运行main()函数开始批量翻译

### 示例代码:
    asyncio.run(main())

### 输入文件格式:
    CSV文件，第一列为中文内容

### 输出文件格式:
    同一CSV文件，第二列为处理结果
